JassBird - for those too lazy to run jass(1) themselves

You can specify multiple users as a comma-separated list of usernames.

You can specify multiple groups as a comma-separated list of group names.

You have to specify at least one user or one group; users and groups may both be specified simultaneously.

You can provide input either via the text field or by uploading a file.
